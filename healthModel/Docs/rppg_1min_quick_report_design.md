# rPPG 1分钟快检报告 — 指标设计方案

## 设计原则

1. **输入边界明确**：仅使用当前 1min rPPG 可计算的 HR 序列指标（无呼吸、无频域、无 BVP）
2. **与5分钟报告差异化**：采用不同的指标命名体系和评分维度，避免用户直接对比数值
3. **短窗口定位**：1分钟定位为"即时状态快照"，不等同于临床级评估
4. **质量门控**：quality_flag = poor 时不输出健康结论

---

## 可用输入指标清单

来源：`rppg_fiber_1min_metrics_comparison.csv`

| 类别 | 指标 | 说明 |
|------|------|------|
| 质量 | valid_ratio, quality_flag, missing_count, hr_outlier_count | 数据可信度 |
| 心率水平 | avg_heart_rate, median_heart_rate, min_heart_rate, max_heart_rate, heart_rate_range | 基础心率 |
| 心率趋势 | heart_rate_slope, heart_rate_delta | 窗口内变化方向 |
| 心率稳定性 | SD_HR, CV_HR | 心率离散度 |
| 代理HRV | MeanJJ, SDNN, RMSSD_HR, HRV_diff | HR推导的伪RR间期指标 |
| Poincaré | SD1, SD2, SD1_SD2_ratio | 短期/长期变异比 |
| 实验性 | PNN50_experimental, SampEn_HR_experimental, VAI_experimental, VLI_experimental, ANS_activity_experimental | 仅供参考 |

## 一、心理健康即时评估（Instant Mental Wellness）

> 5分钟报告使用"焦虑抑郁 / 情绪应激 / 疲劳水平 / 压力敏感度"四维度。
> 1分钟报告改用"即时紧张度 / 即时放松度 / 心率波动敏感性"三维度，强调瞬时状态而非持续评估。

### 1.1 即时紧张度（Instant Tension Index）

- **计算逻辑**：综合 heart_rate_slope（正斜率 → 激活态）、SD_HR（高波动 → 不安定）、avg_heart_rate 偏离个人基线程度
- **输入指标**：heart_rate_slope, SD_HR, avg_heart_rate
- **输出**：0~100 分，越高越紧张
- **分级**：
  - 0~30：放松
  - 31~55：平稳
  - 56~75：轻度紧张
  - 76~100：明显紧张
- **与5分钟差异**：5分钟用"焦虑抑郁"百分比 + SCL90 标签；1分钟仅反映即时激活水平，不做焦虑/抑郁分类

### 1.2 即时放松度（Instant Relaxation Index）

- **计算逻辑**：综合 RMSSD_HR（高 → 副交感活跃 → 放松）、heart_rate_slope（负斜率 → 减速趋势）、SD1_SD2_ratio（高 → 短期变异主导）
- **输入指标**：RMSSD_HR, heart_rate_slope, SD1_SD2_ratio
- **输出**：0~100 分，越高越放松
- **分级**：
  - 0~25：紧绷
  - 26~50：一般
  - 51~75：较放松
  - 76~100：深度放松
- **与5分钟差异**：5分钟无此独立维度；1分钟将放松作为独立正向指标呈现

### 1.3 心率波动敏感性（HR Fluctuation Sensitivity）

- **计算逻辑**：CV_HR 归一化 + HRV_diff 归一化
- **输入指标**：CV_HR, HRV_diff
- **输出**：低 / 中 / 高 三级
- **解读**：
  - 低：心率平稳，情绪稳定或低活动
  - 中：正常波动范围
  - 高：心率敏感波动，可能与情绪应激、体动或环境刺激有关
- **与5分钟差异**：5分钟用"压力敏感度"百分比；1分钟用离散度直接分级，不做压力归因

## 二、运动健康即时评估（Instant Fitness Snapshot）

> 5分钟报告包含 BMI、代谢、心肺功能、运动安全心率区等。
> 1分钟报告无法计算 BMI/代谢（需身高体重），改用"即时心率储备利用率 / 心率恢复趋势 / 运动耐受提示"。

### 2.1 即时心率储备利用率（HR Reserve Utilization）

- **计算逻辑**：`(avg_heart_rate - resting_hr_estimate) / (estimated_max_hr - resting_hr_estimate) × 100%`
  - resting_hr_estimate：取 min_heart_rate 作为近似
  - estimated_max_hr：若有年龄则用 208 - 0.7×age；无年龄则用 185 作为默认
- **输入指标**：avg_heart_rate, min_heart_rate, (年龄可选)
- **输出**：0~100%
- **分级**：
  - 0~20%：静息态
  - 21~50%：轻度活动
  - 51~70%：中度活动
  - 71~85%：高强度
  - \>85%：极限区（提示注意安全）
- **与5分钟差异**：5分钟输出"储备心率"绝对值 + 运动量分区；1分钟输出利用率百分比，侧重即时状态

### 2.2 心率恢复趋势（HR Recovery Tendency）

- **计算逻辑**：heart_rate_delta（后10秒均值 - 前10秒均值）
  - 负值 → 心率在下降（恢复中）
  - 正值 → 心率在上升（激活中）
- **输入指标**：heart_rate_delta, heart_rate_slope
- **输出**：恢复中 / 稳定 / 激活中 三级
- **判定**：
  - heart_rate_delta < -3 且 slope < 0：恢复中
  - |heart_rate_delta| ≤ 3：稳定
  - heart_rate_delta > 3 且 slope > 0：激活中
- **与5分钟差异**：5分钟用"恢复指数"四级评估（较差/一般/良好/优秀）；1分钟仅判断即时方向

### 2.3 运动耐受提示（Exercise Tolerance Hint）

- **计算逻辑**：综合 heart_rate_range（极差大 → 心率调节幅度大）、SDNN（高 → 调节能力好）、avg_heart_rate 水平
- **输入指标**：heart_rate_range, SDNN, avg_heart_rate
- **输出**：良好 / 一般 / 需关注 三级
- **与5分钟差异**：5分钟有"心肺功能"和"锻炼累积"四级评估；1分钟仅给出即时耐受提示

## 三、慢病风险即时筛查（Instant Chronic Risk Screen）

> 5分钟报告分"消化系统 / 代谢系统 / 循环系统"三个子系统，各有风险等级和分类。
> 1分钟报告无法做系统级分类，改用"心血管应激指数 / 代谢负荷提示 / 节律异常提示"三个即时风险信号。

### 3.1 心血管应激指数（CV Stress Signal）

- **计算逻辑**：综合 avg_heart_rate 偏高程度、SD_HR 异常、heart_rate_range 过大
- **输入指标**：avg_heart_rate, SD_HR, heart_rate_range
- **输出**：0~100 分
- **分级**：
  - 0~30：低应激
  - 31~55：正常范围
  - 56~75：轻度应激
  - 76~100：明显应激（建议进一步检查）
- **与5分钟差异**：5分钟有"循环系统风险等级"百分比 + 四分类；1分钟仅反映即时心血管负荷信号

### 3.2 代谢负荷提示（Metabolic Load Hint）

- **计算逻辑**：静息心率偏高（avg_heart_rate > 80 且处于静息态）+ RMSSD_HR 偏低 → 代谢负荷偏高的间接信号
- **输入指标**：avg_heart_rate, RMSSD_HR, heart_rate_slope
- **输出**：正常 / 偏高 / 需关注 三级
- **前提**：仅在判定为静息态（HR Reserve Utilization < 20%）时输出，运动态不适用
- **与5分钟差异**：5分钟有"代谢系统风险"百分比 + 四分类；1分钟仅在静息态给出间接提示

### 3.3 节律异常提示（Rhythm Anomaly Hint）

- **计算逻辑**：hr_outlier_count 偏高 + HRV_diff 异常大 + PNN50_experimental 异常
- **输入指标**：hr_outlier_count, HRV_diff, PNN50_experimental
- **输出**：未见异常 / 轻度波动 / 建议复查 三级
- **与5分钟差异**：5分钟有"心率散点图"域内/域外风险指数 + 四级分类；1分钟无散点图，仅用统计异常检测

## 四、自主神经即时评估（Instant ANS Assessment）

> 5分钟报告有完整的 ANS 7项标准化指数 + 16种分类 + 能量状态。
> 1分钟报告无频域指标，改用"即时自主神经活性 / 即时神经平衡趋势 / 即时调节弹性"三维度。

### 4.1 即时自主神经活性（Instant ANS Activity）

- **计算逻辑**：SDNN 对数映射到 0~100
  - `score = clip(ln(SDNN) / ln(160) × 100, 0, 100)`
- **输入指标**：SDNN
- **输出**：0~100 分
- **分级**：
  - 0~25：活性偏低
  - 26~50：活性一般
  - 51~75：活性良好
  - 76~100：活性旺盛
- **与5分钟差异**：5分钟"ANS活性指数"基于完整 HRV 频域+时域综合建模；1分钟仅用 SDNN 代理，量纲和分值范围不同

### 4.2 即时神经平衡趋势（Instant ANS Balance Tendency）

- **计算逻辑**：SD1_SD2_ratio 钟形映射
  - ratio ≈ 0.3~0.5 时为平衡区
  - ratio 过高 → 短期变异主导（副交感偏强）
  - ratio 过低 → 长期变异主导（交感偏强）
- **输入指标**：SD1_SD2_ratio, SD1, SD2
- **输出**：交感偏强 / 平衡 / 副交感偏强 三级
- **与5分钟差异**：5分钟"ANS平衡指数"用 LF/HF 频域比值 + 钟形映射；1分钟用 Poincaré 比值替代，不可直接比较

### 4.3 即时调节弹性（Instant Regulatory Resilience）

- **计算逻辑**：综合 RMSSD_HR（反映快速调节能力）和 heart_rate_range（反映调节幅度）
  - `score = clip(0.6 × normalize(RMSSD_HR) + 0.4 × normalize(heart_rate_range), 0, 100)`
- **输入指标**：RMSSD_HR, heart_rate_range
- **输出**：0~100 分
- **分级**：
  - 0~30：弹性偏低
  - 31~55：弹性一般
  - 56~75：弹性良好
  - 76~100：弹性优秀
- **与5分钟差异**：5分钟"ANS适应指数"基于多维度综合建模；1分钟仅用时域代理指标

## 五、中医体质即时辨识（Instant TCM Constitution Hint）

> 5分钟报告输出九型体质分类 + 症状分级 + 典型特征。
> 1分钟报告无法做完整体质辨识，改用"体质倾向提示"——基于 HR 特征模式给出 2~3 种可能倾向。

### 5.1 体质倾向提示（Constitution Tendency Hint）

- **计算逻辑**：基于规则映射（非模型），根据 HR 特征组合给出倾向提示

| HR 特征模式 | 体质倾向提示 | 依据 |
|-------------|-------------|------|
| avg_HR 正常 + SD_HR 低 + RMSSD 适中 + slope ≈ 0 | 平和质倾向 | 心率平稳、变异适中、无明显趋势 |
| avg_HR 偏低 + SDNN 偏低 + RMSSD 偏低 + CV_HR 低 | 气虚质倾向 | 心率偏低且变异性不足，提示气血推动力弱 |
| avg_HR 偏低 + heart_rate_range 小 + SD1 低 | 阳虚质倾向 | 心率偏低且短期调节幅度小，提示阳气不足 |
| avg_HR 偏高 + RMSSD 偏低 + SD1_SD2_ratio 低 | 阴虚质倾向 | 心率偏快、副交感调节弱，提示阴液不足虚热内生 |
| avg_HR 正常偏高 + SD_HR 高 + HRV_diff 大 | 痰湿质倾向 | 心率波动大、间差跳跃明显，提示气机运化不畅 |
| avg_HR 偏高 + slope 正 + heart_rate_range 大 | 湿热质倾向 | 心率偏快且持续上升趋势，提示内热偏盛 |
| avg_HR 正常 + RMSSD 偏高 + PNN50 偏高 | 血瘀质倾向 | 副交感过度活跃可能与血行迟缓有关 |
| avg_HR 正常 + SD_HR 高 + slope 波动 + SampEn 高 | 气郁质倾向 | 心率不稳定、复杂度高，提示情志不畅气机郁滞 |

- **输入指标**：avg_heart_rate, SD_HR, RMSSD_HR, SDNN, CV_HR, heart_rate_slope, heart_rate_range, SD1, SD1_SD2_ratio, HRV_diff, PNN50_experimental, SampEn_HR_experimental
- **输出**：
  - 主要倾向：1种体质倾向 + 匹配度（高/中/低）
  - 次要倾向：0~1种体质倾向（可选）
- **分级**：
  - 匹配度高：≥4 项特征符合
  - 匹配度中：3 项特征符合
  - 匹配度低：2 项特征符合（仅供参考）
- **与5分钟差异**：5分钟基于 HRV-体质量表配对数据训练的模型，输出确定性分类 + 症状分级；1分钟仅基于规则映射给出倾向提示，明确标注"仅供参考，非诊断"

### 5.2 体质调养提示（Constitution Care Tip）

- 根据 5.1 的主要倾向，输出 1~2 条简短调养建议
- 内容来源：标准中医九型体质调养指南
- 示例：
  - 气虚质倾向 → "建议适度有氧运动，避免过度劳累"
  - 阴虚质倾向 → "建议避免熬夜，注意补充水分"

## 与5分钟报告的关键差异对照

| 维度 | 5分钟报告 | 1分钟快检 | 差异化策略 |
|------|----------|----------|-----------|
| 心理健康 | 焦虑抑郁(%) + 情绪应激(%) + 疲劳 + 压力 | 即时紧张度 + 即时放松度 + 波动敏感性 | 不做焦虑/抑郁分类，仅反映即时激活/放松状态 |
| 运动健康 | BMI + 代谢 + 心肺功能 + 运动心率区 | 储备利用率 + 恢复趋势 + 耐受提示 | 无身体参数输入，仅用HR动态特征 |
| 慢病风险 | 消化/代谢/循环 三系统风险% + 分类 | 心血管应激 + 代谢负荷 + 节律异常 | 不做系统级归因，仅输出即时风险信号 |
| 自主神经 | 7项标准化指数 + 16种分类 + 能量状态 | ANS活性 + 平衡趋势 + 调节弹性 | 无频域指标，用时域+Poincaré替代，分值体系不同 |
| 中医体质 | 九型分类 + 症状分级 + 模型输出 | 体质倾向提示 + 调养建议 | 规则映射非模型，仅给倾向提示 |

---

## 质量门控规则

| quality_flag | 输出策略 |
|-------------|---------|
| good (valid_ratio ≥ 0.80) | 输出全部5个维度 |
| fair (0.60 ≤ valid_ratio < 0.80) | 输出①②④，③⑤标注"数据不足，仅供参考" |
| poor (valid_ratio < 0.60) | 仅输出数据质量信息，不输出健康结论 |

---

## 实现建议

1. **Phase 1**：实现规则引擎版本，所有评分基于阈值和线性映射
2. **Phase 2**：收集 rPPG + 问卷配对数据，训练轻量分类模型替代规则
3. **Phase 3**：接入 BVP 波形后升级为真实 HRV 指标，提升评估可靠性
